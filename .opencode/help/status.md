# /status - 查看进度

## 描述

显示当前项目状态，已完成的功能列表和待办事项。

## 工作流程

```
用户输入 /status
  ↓
检测项目状态
  ↓
识别已完成阶段
  ↓
识别待办阶段
  ↓
生成进度报告
```

## 状态说明

| 状态 | 描述 |
|------|------|
| 0-1 模式 | 新项目，正在从零开始开发 |
| 迭代模式 | 已有项目，正在添加新功能 |

## 相关指令

| 指令 | 描述 | 关系 |
|------|------|------|
| `/check` | 完整度检查 | 下一步 |
| `/help` | 获取帮助 | 相关 |
