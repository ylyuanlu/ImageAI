# AI 开发流程帮助菜单

## 快速开始

| # | 指令 | 描述 | 前置条件 |
|---|---|------|------|----------|
| 1 | `/do` | 一键智能执行（全流程） | 无 |
| 2 | `/prd` | 开始需求收集 | 无 |
| 3 | `/ui` | 生成 UI 提示词 | Product-Spec.md |
| 4 | `/art` | 生成设计稿 | UI-Prompts.md |
| 5 | `/confirm` | 设计确认 | design-assets/ |
| 6 | `/dev` | 开始代码开发 | Product-Spec.md |

## 查看状态

| # | 指令 | 描述 |
|---|---|------|------|
| 7 | `/status` | 查看项目进度 |
| 8 | `/check` | 检查功能完整度 |

## 质量保障

| # | 指令 | 描述 | 前置条件 |
|---|---|------|------|----------|
| 9 | `/run` | 本地运行验证 | 代码已实现 |
| 10 | `/test` | 执行单元测试 | 代码已实现 |
| 11 | `/review` | 代码审查 | 测试通过 |
| 12 | `/fix` | 快速修复代码问题 | 代码已实现 |
| 13 | `/deploy` | 部署上线 | 审查通过 |

## 帮助

| # | 指令 | 描述 |
|---|---|------|------|
| 14 | `/help` | 显示此帮助 |
| 15 | `/help <指令>` | 查看指令详情 |

## 完整工作流

```
/prd → /ui → /art → /confirm → /dev → /run → /test → /review → /deploy → /check
```

## 获取帮助

- 输入 `/help` 查看帮助菜单
- 输入 `/help <数字>` 查看指令详情
- 输入 `/status` 查看当前项目状态
